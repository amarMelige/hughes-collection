# Hughes Estate Collection

This project is a private, password-protected digital catalog for the Hughes Estate's personal collection of firearms and vehicles.

## Overview

The site provides a searchable, categorized, and visually rich presentation of an extensive collection of firearms (pistols, revolvers, rifles, shotguns, air guns, and black powder) and vehicles (cars, trucks, motorcycles). Its main features include:

- **Password-protected access**: Only authorized viewers can browse the collection.
- **Detailed catalog**: Each item displays make, model, caliber/year, serial number, descriptive notes, and (where available) images.
- **Categorization and statistics**: Quickly browse by item type (firearms, vehicles, sold items, and subcategories) and view collection statistics (totals, asking price, and value estimates).
- **Search and sort**: Instantly filter by make, model, caliber, and more, and sort items by number, value, or sale status.
- **Gallery and modal views**: View detailed popups and image galleries for each item.

## Usage

1. Open the site (the default entry point is `index.html`).
2. Enter the password to unlock access.
3. Search, browse, and select items to view details and images.

## Structure

- `index.html`: Main application logic and UI
- `images/`: Item images
- All logic, catalog data, and visual rendering are self-contained in the HTML file for privacy and portability
